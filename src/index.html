The Block Puzzle Solver calculates the solutions to monomino, domino, triomino, tetromino and pentomino(1) based rectangular or square tangram puzzle in a user-defined area, with support for block rotation and reflection. The fast version uses a random unique block combinations. The slow version tests all possible block combinations. The plan is to include more pentominoes, heptominoes, octominoes & other polyominoes.
